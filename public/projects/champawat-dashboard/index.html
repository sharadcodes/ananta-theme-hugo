    Adarsh Champawat Geospatial Dashboard

    • Jan 15, 2024 – Dec 1, 2024
    • 1 min read
    • Completed
    Repository Live demo Documentation

    Technology

    Web GIS PostGIS GeoServer Svelte LLM RAG

    Overview

    The Adarsh Champawat Geospatial Dashboard is a comprehensive Web GIS application designed for district-level spatial analysis and decision support. Developed for Champawat District in Uttarakhand, India, the dashboard integrates advanced geospatial analysis with AI/ML capabilities.

    Key Features

    • Interactive map visualization with multiple layers
    • Real-time spatial data analysis
    • AI-powered insights using LLM + RAG
    • Custom report generation
    • Multi-user collaboration features
    • Mobile-responsive design

    Technical Architecture

    The system uses a modern stack combining:

    • Backend: REST APIs with PostGIS for spatial queries
    • Frontend: Svelte for reactive UI components
    • Geospatial: GeoServer for WMS/WFS services
    • AI Integration: LLM + RAG for intelligent analysis
    • Deployment: Docker containers on EC2/ECS

    Impact

    The dashboard has been adopted by district administration for:

    • Land use planning and management
    • Environmental monitoring
    • Disaster response coordination
    • Resource allocation optimization

    Lessons Learned

    This project demonstrated the value of combining traditional GIS capabilities with modern AI techniques. Key insights include the importance of user-centered design for domain experts and the need for robust data quality assurance in geospatial applications.
